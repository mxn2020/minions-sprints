# Minions Sprints

Sprint definitions, scope, and velocity tracking

## Overview

This project is part of the [Minions ecosystem](https://github.com/mxn2020/minions) and provides:

- **TypeScript SDK** (`@minions-sprints/sdk`) — Core library for Node.js
- **Python SDK** (`minions-sprints`) — Python bindings
- **CLI Tool** (`@minions-sprints/cli`) — Command-line interface
- **Web App** — Interactive playground at [sprints.minions.wtf](https://sprints.minions.wtf)
