---
title: Installation
description: Install Minions Sprints packages
---

## TypeScript / Node.js

```bash
npm install @minions-sprints/sdk minions-sdk
```

## Python

```bash
pip install minions-sprints
```

## CLI

```bash
npm install -g @minions-sprints/cli
```

## Requirements

- **Node.js** 20 or later
- **Python** 3.11 or later (for Python SDK)
